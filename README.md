# confettysh

## Installation

```sh
go install github.com/charmbracelet/confettysh@latest
```


## Usage

Confetti and Fireworks over SSH.

```sh
confettysh
```

```sh
ssh -p2222 localhost

ssh -p2222 localhost -t confetti
ssh -p2222 localhost -t fireworks
```
